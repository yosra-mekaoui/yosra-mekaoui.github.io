# Dossier pour votre CV

Ce dossier est destiné à contenir votre CV au format PDF.

## Comment ajouter votre CV

1. Préparez votre CV au format PDF
2. Nommez-le `Yosra_Mekaoui_CV.pdf`
3. Placez-le dans ce dossier

Une fois que votre CV est ajouté, les visiteurs de votre site pourront le télécharger en cliquant sur le bouton "Télécharger mon CV" dans la section Contact.

## Note

Si vous souhaitez utiliser un nom différent pour votre fichier CV, assurez-vous de mettre à jour le lien dans le fichier `index.html` :

```html
<a href="cv/VOTRE_NOUVEAU_NOM_DE_FICHIER.pdf" download class="btn secondary-btn">
    <i class="fas fa-download"></i> Télécharger mon CV
</a>
```
